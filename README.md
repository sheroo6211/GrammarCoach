# GrammarCoach

Free AI-powered grammar checker and writing improver. No login or installation needed. Works in any browser.

## Features
- Fix grammar and spelling instantly
- AI rewrites your text to sound professional
- 3 tones — Professional, Casual, Academic
- Corrections explained
- Vocabulary enhancer
